# Tarot Card Image Assets｜塔罗牌图片资源

本项目默认从以下路径加载牌面图片：

```text
assets/cards/{card.id}.jpg
```

如果图片文件不存在、加载失败或超时，页面会自动显示复古占位牌面，不会报错。

建议使用已获得授权、公共领域或个人自备的 Rider-Waite-Smith / 塔罗牌图片。不要直接热链不稳定的远程图片。为了 GitHub Pages 稳定展示，推荐把图片下载后放入 `assets/cards/` 目录。
